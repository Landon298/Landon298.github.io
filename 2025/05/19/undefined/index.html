前言

注：

符号 含义
p, q 两个大素数
n = p * q 模数
φ(n) = (p-1)(q-1) 欧拉函数
e 公钥指数（一般选 65537）
d 私钥指数，使得 e * d ≡ 1 (mod φ(n))
m 明文（message），满足 0 ≤ m < n
c 密文（ciphertext）

总结口诀：

有 d 求 m，有 e 求 d；有 p,q 就能解密。
