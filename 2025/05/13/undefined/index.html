前言

不参加ctf比赛真的是不知道自己是有多么的菜，更严重的是之前做过的题由于没有总结归纳导致再次碰到同样类型的题仍然还是不会，可见总结归纳是多么重要。因此，作为一名ctf萌新，我决定改掉这种不好的习惯，每次做题后都要写下该题的wp,痛改前非，接下来让我们一起进步吧。

1.old-fashion

题目信息

  • 类别： Crypto
  • 题目描述： 注意：得到的 flag 请包上 flag{} 提交
  • 附件下载： ec8bd2cf-5fa6-4fed-9d8f-f820bae6cfd6.zip

解题思路、步骤：

首先下载附件，解压打开后出现题目.txt文件，打开后内容出现：

观察后发现后面的f1_2jyf-k3_jg1-vb-vl_l应该是flag，于是采用凯撒进行枚举，没有发现flag，然后想到 https://quipqiup.com/ 网站(一个快速且自动化的密码求解器)，观察可知dsln应该是flag，所以在Clues这一栏中填写dsln=flag，然后点击solve，即可得到flag。

2.[AFCTF2018]Morse

题目信息

  • 类别： Crypto
  • 题目描述： 得到的 flag 请包上 flag{} 提交。
  • 附件下载： attachment.txt

解题思路、步骤：

首先下载附件，打开后出现一段密码，立即知道是摩斯密码

于是打开随波逐流软件，选择摩斯密码解密，得到一串密码

根据形式可以知道，应该是base x编码，于是直接一键解码，得到flag。

3.隐藏的钥匙

题目信息

  • 类别： Misc
  • 题目描述： 路飞一行人千辛万苦来到了伟大航道的终点，找到了传说中的One piece，但是需要钥匙才能打开One Piece大门，钥匙就隐藏在下面的图片中，聪明的你能帮路飞拿到钥匙，打开One Piece的大门吗？ 注意：得到的 flag 请包上 flag{} 提交
  • 附件下载： 8f6577ab-4019-4e61-aeec-a6d4fa492994.zip

解题思路、步骤：

首先下载附件，解压后出现一张.jpg照片，照片标题为“隐藏的钥匙”，根据之前做图像类Misc的习惯，放到winhex中查看，并且直接ctrl+f搜索flag，发现flag在右边

根据提示直接用base64解码即可得到flag。（但是如果选择一键解码，则会得不到flag）

4.另外一个世界

题目信息

  • 类别： Misc
  • 题目描述： 注意：得到的 flag 请包上 flag{} 提交
  • 附件下载： 31c48bab-cf07-4588-bc34-bd677811cf20.zip

解题思路、步骤：

首先下载附件，解压后出现一张.jpg文件，直接放到winhex中查看，查找flag发现没有，然后观察开头和结尾，发现结尾处有一串二进制密码

然后在浏览器搜索二进制转字符串在线工具，把原码放进去即可得到flag。

5.[极客大挑战 2019]Secret File

题目信息

  • 类别： Web
  • 题目描述： 无

解题思路、步骤：

启动靶机，发现是一个网站，直接ctrl+u查看源代码，发现里面有一个./Archive_room.php文件，于是直接打开

直接点击SECRET，页面如下

尝试几次后，想到应该是有一个网页闪过去了，于是直接抓包，打开bp,开始抓包

最终在Response中发现了一个注释掉的一个文件，然后直接在靶机上打开，发现出现了一堆代码，但是没有flag信息，于是这时候解酒药想到代码审计

这时候出现了一串base64的编码，直接放到工具中解码，然后可得到flag。
