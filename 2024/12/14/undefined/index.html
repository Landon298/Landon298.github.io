前言

下面是本萌新第一次完整的写下全部的ctf题，之前练的一直都是web方向的，但是通过在这次测试使我学到了许多misc和crypto的知识。如有错误，欢迎大家来与我交流。

题目一

题目信息

  • 题目名称： miao~
  • 类别： Misc
  • 题目描述： 猫猫真的很可爱捏~
  • 附件下载： miao~附件.zip

第1题：

解题思路、步骤：

打开附件发现里面有一张以jpg为后缀的照片，打开后发现没有有用的信息，于是用到了foremost软件，将这张照片丢到foremost文件夹里面，在生成的output文件夹里面发现了一段录音，打开后是猫叫，于是打开在线音频分析工具，显示波形图得到了一个类似密码CatCTF,然后想到了DeepSound能分析音频，于是打开后把那个音频丢了进去，果然要求输入密码，输入CatCTF后发现出来了一个flag.txt,于是打开DeepSound文件夹里的flag.txt,发现里面是猫叫的汉字，正好之前看了一个csdn上的：https://blog.csdn.net/Aluxian_/article/details/141812967 ，正好里面有一个兽语解码，网址为：http://hi.pcmoe.net/roar.html .把这个输进去后，就得到了flag.

Flag:

CatCTF

题目二

题目信息

  • 题目名称： misc1
  • 类别： Misc
  • 题目描述： 出题人没有给题目描述哦
  • 附件下载： 为什么会打不开呢.ppt

第2题；

解题思路、步骤：

首先下载附件，打开后只有一句话，然后尝试用010 Editor分析，分析后搜索发现一个假的flag:flag{Not_So_Easy_Stupid_Boy},于是再查看16进制前面的英文，发现[Content_Types]，询问chatgpt后得出这是一个zip文件，于是改后缀为zip,发现里面有很多文件，在里面发现有一个flag文件夹，打开后里面有一个flag.txt,打开后又得到一个假的flag:flag{Not_So_Easy_Stupid_Boy},这时候已经红温了。这时候打开没用的东西这一个文件夹，发现里面有一个ex.txt,打开后发现：我忘记这个软盘该怎么打开了。瞬间感觉在哪里见过。同时又打开file文件夹，发现里面有一个file文件，打开后显示乱码，再用010 Editor打开在开头发现了CTF.SHOW,这时候终于想起来了这道题做过，原题目在ctfshow的misc的misc2,于是照着过程以及之前写的记忆写了一遍：新建虚拟机，在软盘驱动器处选择使用软盘映像文件，选择那个file文件，最后打开后出现了这个flag.

(还是刷题多好啊）

题目三

题目信息

  • 题目名称： misc2
  • 类别： Misc
  • 题目描述： 呆在猪圈里也要会使用 foremost 噢
  • 附件下载： aihe.zip

第3题：

解题思路、步骤：

首先看题目有点懵逼，然后下载了这个附件，打开后发现是一个音频，但是却有一张照片，想到提示：foremost，于是把这个音频拖到了那里，最后得到了一张照片。很明显能看出这张照片下面被隐藏了一部分，查看属性发现高度很小才371，于是用010 Editor 打开，同时查询网上怎样调整高度，把鼠标放到16进制(0:00A0,3)处，发现01详细为371，然后将这个改为02（看到旁边都是2，蒙的），保存后查看照片，发现下边隐藏的出现了，出现一堆奇怪字符还是手写，问了chargpt,它解不出来于是又打开了https://blog.csdn.net/Aluxian_/article/details/141812967，发现里面有一个猪圈密码，用解密网站：http://moersima.00cha.net/zhuquan.asp，解码后得出flag.，不过要加flag{}.

Flag:

flag

题目四

题目信息

  • 题目名称： 不确定，再看看
  • 类别： Misc
  • 题目描述： 做题做累了吧，给你准备了一道钢琴曲，要仔细听哦！我藏得很深。
  • 附件下载： 不确定，再看看

第4题：

解题思路、步骤：

首先下载附件，打开后是一个百度网盘文件，于是下了半天，最后得到了一个zip文件，里面有一个音频文件，放到在线音频分析工具里面，没有结果。然后尝试用010 Editor 打开发现里面有敏感字符：base64.txt,然后用DeepSound发现里面有一个base64.txt文件，打开发现是一个base64编码的，然后猜测是base64隐写，然后在网上找脚本，跑出来后得到flag。

Flag:

flag

题目五

题目信息

  • 题目名称： 扫描二维码
  • 类别： Misc
  • 题目描述：
  • 附件下载： pic(4).png

第5题：

解题思路、步骤：

下载附件发现是一个二维码，然后在网上找在线二维码扫描工具，结果是一些二进制，然后丢给chatgpt,它先是转为十进制再转为ASCII码，发现直接解出来了一个类似于flag,但是最后是错的。然后我仔细研究后才发现一个字母后面跟一个数字，于是我让chatgpt去整合，没想到它坑我，直到我自己亲自去掉数字才知道它把falg改为flag,最后也提交成功了。

Flag:

falg

题目六

题目信息

  • 题目名称： 旗是什么？
  • 类别： Misc
  • 题目描述： 米奇回到了米奇妙妙屋，这时候需要念出奇妙的咒语 “米斯卡，慕斯卡，米老鼠！” 但是没有起作用，这时，天上飘来一张纸，上面写着： 公正文明和谐文明法治富强和谐民主公正自由平等爱国自由文明法治 民主公正文明和谐和谐自由文明公正敬业公正和谐公正友善爱国自由 诚信富强公正敬业平等敬业和谐文明平等诚信富强法治民主公正和谐 平等民主和谐友善法治和谐诚信和谐
  • 附件下载： 无

第6题：

解题思路、步骤：

这是一道纯粹的解密题，在看到一堆社会主义核心价值观，幸亏之前做过这道题，于是用社会主义核心价值观解码工具解码，解码后发现是一个base64解码，再进行解码，解码后出现了一个全是字母的密码，这时已经有点蒙了，于是问了问chatgpt，得到了一些解决方法，于是一个一个试。最后用凯撒密码暴力解码偏移9得到了一个比较像的密码，这时候又蒙了，于是在一番苦想后得到了一个可能的解码方式，栅栏解码，于是再次暴力尝试，最后终于得到了flag.

有看到很多网友关心 觉得玩杰哥梗会不会感到不适 跟大家解释一下

现在这角色，在几年前已经打从心底接受 杰哥也是我人生的一部分

很开心很幸运 演到这角色 杰哥这名字让我认识了很多人 交了很多朋友

现在我自己都会玩这个梗 人在世上开心快乐最重要 这世界需要欢笑 来温暖 辛苦的大家

能在网路世界用这角色跟大家互动 我觉得很开心 所以想玩梗叫我杰哥 或是 铁牛都可以的 这些称呼，都是我！ 不就是称呼而已大家开心就好

  • 附件下载： 外部链接

第7题：

解题思路、步骤：

刚开始点击链接进去：是一个关于杰哥的视频，第一个想法flag藏在弹幕里，但是当我看完后发现没有，然后我就用了web的解题方法，检查页面，发现也没线索，于是就废弃了这道题。之后，再次打开这道题，发现网址似乎有些不对，反正比赛也快结束了，于是抱着试一试的心态截取了BV1rA411g7q8，在经过一番尝试后得到了flag,真是没想到。

  • 题目描述： 无
  • 附件下载： 爆破哦附件

第8题：

解题思路、步骤：

在看到题目后我就知道要用爆破了，回想起招新赛学长讲的题，拿出了许久不用的Advanced Archive Password Recovery，于是把这个文件放了进去，结果得到了一个密码：fish,然后想到这个文件可能有问题，于是用010 Editor 打开发现敏感词flag.txt，于是更改后缀名为txt,发现里面有个文件但是需要密码，用fish打开后得到一个密码，然后用base64解码解出flag.

题目十

题目信息

  • 题目名称： 不止加密一次哦
  • 类别： Crypto
  • 题目描述： 是一种食物哦,答案用小写
  • 附件下载： 不止加密一次哦附件.txt

第10题;

解题思路、步骤：

下载附件，打开后是一串摩斯密码，解密后，是BAABAAABBBABAAABAAABABAAABAAABAAAAAABBAABAAABBABAAAABAABAAAA，然后想到提示的食物https://blog.csdn.net/Aluxian_/article/details/141812967，发现也没有，于是问浏览器，浏览器中有一种解密方式正好可以：培根密码，解出后为THISISANSWER,转为小写：thisisanswer，解出flag
